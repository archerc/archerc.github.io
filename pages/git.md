# Git 用法与技巧

## 命令行方式

列出所有设置

> git config -l

列出所有全局配置文件中的配置

> git config -l --global

列出所有系统配置文件中的配置

> git config -l --system


列出所有当前仓库配置文件中的本地配置

> git config -l --local


设置别名

> git config alias.st status
git config alias.ci commit
git config alias.br branch

## 客户端

推荐使用[SourceTree](https://www.sourcetreeapp.com/)客户端，图形界面，支持跨平台，多语言。
